iproute2 · Linux 網路管理

一個指令，取代 ifconfig、route、arp

ip 是 iproute2 套件中的網路管理工具，統一管理 IP 位址、路由表、網卡與 ARP 快取——舊時代分散的三個指令，現在都收進同一支工具裡。

root@serverbash
$

Object

四大管理物件

語法固定為 ip [Object] [Command]，先認識四個物件，後面的指令都是它們的延伸。

address 簡寫 a

管理主機上的 IP 位址——新增、刪除、查看。

route 簡寫 r

管理路由表，決定封包該往哪個閘道 / 介面送出。

link 簡寫 l

管理網路卡（NIC）本身，例如啟用或停用介面。

neigh 簡寫 n

管理 ARP / Neighbor Table，也就是 IP 對應 MAC 的快取。

01 · Address

IP 位址管理

新增、刪除、查看主機介面上的 IP 位址。

02 · Route

路由管理

控制封包的下一步：預設閘道與靜態路由。

03 · Link

網卡管理

查看、啟用、停用網路介面。

04 · Neigh

ARP / Neighbor Table

查看區域網路中 IP 對應 MAC 的快取表。

Tip

推薦查看方式

彩色 + 精簡輸出

$ ip -c -br a
  • -c　彩色顯示，介面狀態一眼可辨
  • -br　簡潔模式（brief），每個介面一行
